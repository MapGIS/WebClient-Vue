Leaflet TimeDimension example 18

Integration with leaflet-velocity

Information

leaflet-velocity is a plugin for Leaflet that creates a canvas visualisation layer for direction and intensity of arbitrary velocities. In this example, we are using SOCIB Data API to get the values of sea water velocity measured by a High Frequency Radar (HFR) to feed a leaflet-velocity layer that is updated on every time change.

This example use an extra Leaflet.TimeDimension class: L.TimeDimension.Layer.VelocityLayer.
