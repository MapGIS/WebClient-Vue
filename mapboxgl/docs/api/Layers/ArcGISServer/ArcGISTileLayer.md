# ArcGISTileLayer

## Props

All common [layers props](/zh/api/Layers/README.md#props)

### `baseUrl`

- **类型:** `String`
- **默认值:** `null`
- **Non-Synced**
- **描述:** 地图请求的基地址路径。

### `tileSize`

- **类型:** `Number`
- **默认值:** `256`
- **Non-Synced**
- **描述:** 若没传递该参数，则默认为地图坐标系。

### `zoomOffset`

- **类型:** `Number`
- **默认值:** `0`
- **Non-Synced**
- **描述:** 层级偏移量
