# 模块说明
webclient-vue-mapboxgl 主要包含以下模块：</br>

- 地图（主模块）</br>
- 地图子组件（绘制、量测、卷帘、气泡等工具类）</br>
- 图层（OGC、IGServer、ArcGIS、矢量、栅格等相关图层）</br>
- 可视化（echarts、mapv、建筑白膜、轨迹跟踪等）
- 服务（综合查询）
